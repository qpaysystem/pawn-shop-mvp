Отчёт 1С для импорта товаров скупки
=====================================

Команда: php artisan lmb:import-purchase-items-from-1c-report storage/app/1c_purchase_report.tsv

Файл 1c_purchase_report.tsv — TSV (колонки через табуляцию), как в выгрузке 1С:
  Колонка 1: Дата (04 марта, 11 марта)
  Колонка 2: Тип операции + номер + дата (должно содержать «Скупка» и номер 00БП-XXXXXX)
  Колонка 3: Точка (адрес магазина)
  Колонка 4: Клиент (ФИО)
  Колонка 5: Наименование товара (Цепь, Золото 585, 8,67 гр. и т.д.)
  Колонка 6: Сумма

Скопируйте из Excel/1С в этот файл (сохраните как TSV или вставьте с табами между колонками).
